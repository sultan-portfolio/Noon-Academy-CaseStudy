Guardrails & Anti-Gaming Protocols

Bold actions require sharp checks. Every system designed to be measured is a system designed to be gamed.

01. Data Validation

Attempt Integrity: Filter for Genuine Cognitive Effort before flagging.
Level Normalization: Difficulty isn't linear. Triggers must be dynamic.
Metric Validity: Correlate 'Trust' with LTV before using it as a proxy.

02. Operational Assumptions

Tech Scalability: Stack must support Real-time Event Triggers.
Facilitator Agency: Assumes teachers have pedagogical training to act on visibility data.
School B: Used as the Golden Standard benchmark.

03. Goodhart’s Law Defense

The Cobra Effect: Independent Spot-Checks to prevent grade inflation.
The Efficiency Trap: Pair LGpE with a Challenge Index (efficient mastery of hard concepts).
Human Friction: Frame alerts as "Support Sprints" to maintain trust.
